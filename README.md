LDML Test keyboard
==============

Description
-----------
Test Keyboard for LDML

You can test this out, if you type:  `1` (enter) `2` (enter) `3` (enter) `4` (enter) it should output:

```
1:basic:OK
2:simple_transform:OK
3:simple_marker:OK
4:simple_reorder:OK
```
